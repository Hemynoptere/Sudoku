Mode d'emploi Sudoku

Cette application a pour but l'apprentissage du sudoku pour des enfants.

L'application permet de créer des grilles de sudoku de différents niveaux de difficulté.
L'application permet de saisir des chiffres en cliquant sur les cases vides.
L'application détecte automatiquement les doublons quand 'Simples' n'est pas coché.
L'application montre pas à pas comment résoudre une grille du début à la fin en cochant certains boutons (Simples et Fil d'Ariane) ou demander une aide momentanée en cochant d'autres boutons (Paires, Doubles, Triples, Lignes, Colonnes). Ces aides peuvent être arrêtées ou reprises en cours de route.

MENUS :

En haut :
- Facile/Moyen/Difficile/Expert → Choisir le niveau de difficulté
- Créer → Génère une nouvelle grille avec le niveau sélectionné
- Vérifier → Affiche les erreurs en rouge puis les efface automatiquement au bout de deux secondes
- Résoudre → Affiche la grille résolue
- Vider → Efface tous les chiffres saisis. Garde les chiffres de la grille d'origine
- Vierge → Génère une grille vierge dans laquelle on peut saisir les données d'une grille existante.Le bouton devient "Fixer" pour valider la grille et jouer comme avec une grille generée. 

En bas :
- Simples → Affiche l'un après l'autre un chiffre solution en vert et en rose les chiffres qui montrent pourquoi cette solution est la bonne (série complète de 1 à 9). 
  Sans 'Simples' coché : les doublons s'affichent en rouge 2 secondes puis s'effacent automatiquement.

- Paires → Affiche un seul chiffre possible entre deux cases contiguës
- Doubles → Affiche les deux seuls chiffres possibles dans deux cases contiguës
- Triples → Affiche les trois seuls chiffres possibles dans trois cases contiguës
- Lignes → Affiche les chiffres candidats pour chaque ligne à gauche
- Colonnes → Affiche les chiffres candidats pour chaque colonne en haut
- Fil d'Ariane → Quand plus aucun simple ne peut être trouvé, affiche plusieurs chiffres possibles dans une case entourée d'un cercle violet. Un bouton supplémentaire "Résoudre Fil" apparaît en haut pour révéler la bonne valeur.
